# PiHole-Panel
PiHole-Panel is a statistics and control panel for Pi-hole and optimised for the Gnome desktop.

![](pihole-panel_preview.png)

PiHole-Panel 1.1

Install/Update:
```
sudo apt install python3-gi
```
```
sudo sh ./setup.sh
```

Uninstall:
```
sudo sh ./uninstall.sh
```

Troubleshoot:
```
rm ~/.config/pihole_panel_configs.xml
```
